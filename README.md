# Neuromorph

This is story-driven quest with dialogue option crafting mechanic made in C# and Unity
Unity version: 2022.3.62f2

https://github.com/user-attachments/assets/a6f0f666-624e-4f39-b760-233483de958e

## Mechanics
* A dialog system where options are unavailable until the required keyword is provided
* You can combine the found words to get new keywords
* Mouse click movement
